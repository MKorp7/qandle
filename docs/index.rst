Welcome to qandle's documentation!
==================================

**QANDLE** is a fast and simple quantum state-vector simulator for hybrid machine learning using the PyTorch library.

.. note::

   This project is under active development.


Installation
------------
To install the latest version of **qandle**, you can use pip:

.. code-block:: console

   $ pip install qandle
